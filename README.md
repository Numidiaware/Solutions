## Project Description

Numidiaware develops powerful, lightweight utilities for Windows users. Our suite focuses on solving specific workflow frustrations through intuitive design and seamless OS integration. Our flagship software includes:

*   **Folder Comments**: A productivity tool that allows you to attach custom text comments and notes to directly to any Windows directory. Avoid losing context—instantly see your notes on hover (tooltip) or in the Windows Explorer details pane without opening hidden text files.
*   **PC Control Suite**: A comprehensive automation and power-management toolkit. It empowers users to schedule complex system tasks, manage shutdowns/restarts precisely, and optimize PC operations from a unified dashboard.

## Requirements / Prerequisites

To run Numidiaware desktop applications, your system must meet the following baseline requirements:
*   **Operating System**: Windows 10 or Windows 11.
*   **Permissions**: Administrator privileges may be required for initial installation and certain system-level features in PC Control Suite.

## Usage Examples

### Using Folder Comments
1. Right-click any folder in Windows Explorer.
2. Select **"Add/Edit Comment"** from the context menu.
3. Type your note (e.g., *"Client assets for Q3. DO NOT DELETE."*) and save.
4. Hover over the folder at any time to instantly read your securely attached note.

### Using PC Control Suite
1. Open the PC Control Suite dashboard.
2. Navigate to the **"Task Scheduler"** tab.
3. Set a timer (e.g., *"Shutdown in 2 hours"*) or define a condition (e.g., *"Sleep when CPU usage drops below 5%"*).
4. Click **Apply**. The suite handles the rest silently in the background.
